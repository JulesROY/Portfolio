Bienvenue sur mon portfolio en ligne !

Dans cette partie, vous allez découvrir les coulisses de mon site. Étant seulement au lycée en classe de terminale, je n’ai aucune expérience professionnelle en développement Web. Mais considérez que ce que vous avez sous les yeux est un exemple de mes capacités, capacités que j’ai acquises en autodidacte, notamment grâce au site Openclassrooms. J’ai développé ce site avec les langages HTML, CSS, et JavaScript (jQuery), en utilisant Visual Studio Code sous Mac OS X. Le site contient différentes parties: une partie contenant mon CV, une autre présentant mes projets, et une partie comportant les documents annexes téléchargeables (diplômes, attestation de stage…). J'ai également developpé mon site en responsive design, c'est-à-dire qu'il comporte une version adapté pour les smartphones. A tout moment vous pouvez me contacter via le lien "Me contacter"

Bonne découverte !
